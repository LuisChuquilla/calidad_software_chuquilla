# COMANDOS GIT

Visualizar status de cambios

```
git status 

```
Incluir al Stage

```

git add .
git add archivo_especifico.py

```
Comprometer o sleccionar cambios para subir al repositorio remoto

```
git commit -m 'comentario'

```

Configuracions básicas

```
git config --global user.name "Luis Chuquilla"
git config --global user.email "luisf.chuquilla@ute.edu.ec"
git config --global core.editor "code --wait"
git config --global color.ui auto
git config --global init.defaultBranch main

```
Consultar configuraciones

```
git --list
git config user.name
git config user.email

```
